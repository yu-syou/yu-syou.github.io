formを使うときはformタグをかき、form内を全てformタグで囲います

  • 主な属性は
  • action 　　データの送信先ページを指定
  • method 　データの転送方法の指定(大体getかpost)
  • name 　　フォームの名前を指定

inputタグにtype属性でtextを指定すると、1行入力欄を使えます

名前：

またplaceholder属性を付与することで、あらかじめテキストを用意できます

名前：

１行テキストの属性は様々なものがあります

  • text 　　　１行のテキスト(初期値)
  • search 　　検索する時のテキスト
  • email 　　　メールアドレス
  • tel 　　　　電話番号
  • url 　　　　WebサイトのURL

複数の選択肢から一つだけを選んで欲しい場合は、ラジオボタンを用意します

主な属性は

  • name 　　ラジオボタンの名前
  • value 　 　送信される選択肢の値
  • checked 　最初から選択されている状態にする時に指定
性別： 男 女 その他

-----------------------------------------------------------------

チェックボックスは選択肢を複数選んで欲しい場合に使います

主な属性はラジオボタンと一緒です

好きな色： 赤 青 黄 緑

また、labelタグ、for属性とid属性を組み合わせることでボタンの名前に選択肢の判定を付与することができます

-----------------------------------------------------------------

type属性にsubmitを指定すると、フォームに入力したものを送信できます

主な属性は

  • name 　 ボタンの名前
  • value 　 ボタンに表示されるテキスト

またtype属性にimageを指定し、src、alt属性をつけることでボタンに画像を使うことができます

-----------------------------------------------------------------

selectタグとoptionタグで選択肢を表示できます

選択肢全体をselectタグで囲み、選択項目はoptionタグでそれぞれ囲みます

selectタグの主な属性は

  • name 　 　セレクトボックスの名前
  • maltiple 　 Shifまたはctrlキー(Macなら⌘キー)で複数の項目を選択可能にする

optionタグの主な属性は

  • value 　　 送信する値
  • selected 　最初から選択されている状態にする時に指定

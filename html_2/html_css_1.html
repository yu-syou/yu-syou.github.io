CSSを適用する

htmlは文章や写真を取り扱うことに対して、CSSは文字の色や画像の装飾などを行うことができます。
CSSで装飾することによって、鮮やかなページを作ることができます。

cssの適用方法について

その１ CSSファイルをhtmlに読み込む

head内で rel属性にstylesheetを指定し、href属性にCSSファイルを指定すれば適用できます

こうすることで複数のhtmlファイルがCSSファイルを読み込めるので、基本的にはこの方法が良いです


その２　htmlのhead内にstyleタグで指定する

head内で、styleタグで囲みCSSを書くことで適用できます

他のhtmlファイルには適用されないので、そのページだけ変えたい時など、場合によって使い分けてください


その3　htmlのタグに直接CSSを書く

このように直接タグ内にCSSを書き込むことでCSSを適用できます

書き換えるのに時間がかかりますが適用順位が上記より高いため、一部分だけ変更するときに使えます

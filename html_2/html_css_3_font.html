猫の1日

ひたすら寝ています

猫は毎日12~16時間は睡眠を取ると言われています。ただし、熟睡している時間は意外と少なく、ほとんどが浅い眠りです。物音がするとすぐ目を覚ますのがそのせいなんですね。


上の文はフォントの種類を変えています

指定方法は
１.フォント名を入れる　　フォント名が日本語である場合やスペースが含まれている時には、""で囲みましょう
２.キーワードを使う　　　sans-serif(ゴシック体)　　serif(明朝体) 　cursive(筆記体) 　fantasy(装飾系) 　monospace(等幅)など
これらは先に指定したものから適用されます



フォント例を示しておきます

明朝体

かしこまったデザインの印象を与え、線の太さに強弱があり、ゴシック体に比べて細い書体が多いので長文に適しています
代表：　ヒラギノ明朝Pro 　　游明朝体 　　はんなり明朝

ゴシック体

装飾が少ないので、様々なデザインに合わせやすいです。見出しやタイトル、短い文などに適しています
代表：　arial black 　　arial norrow 　　Tahoma

装飾系フォント

装飾を目的としているので、デザインに起用されることが多いです。読みにくくなることもあるので、本文に使うのは避けましょう
代表：　abcdefghijklmn

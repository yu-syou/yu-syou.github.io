classとid

同じページ内で同じタグを使うけど、装飾は変えたい場面があります。そんな時は、class属性とid属性をタグに記述してCSSと紐付けましょう。


class

クラスを使う場合、HTMLファイルでタグにclass属性を追記し、任意のクラス名を記述します。CSSファイルには.class名と書きます。下記は同じpタグで囲っていますが、下のpタグにはclassを割り振っています。

クラスを指定しない場合には黒色でテキストを表示

クラスを指定した部分は青色で表示


id

idを使う場合、HTML内に記述する方法はclassと同じです。CSSファイルには#id名と書きます。

idを指定していない部分は黒色で表示

idを指定している部分はオレンジで表示

また、タグ名.class名(またはタグ名#id名)と記述することで、そのクラスが付いた特定のタグにのみ反映されます。


複数のタグやクラスをつける

1つのタグに複数のクラスやIDをつけたい場合は、""内に半角スペースで区切って書きます。

IDとクラスの違い

IDとクラスの違いは何かというと、ページ内で同じ名前のIDは使えず、クラスは何度でも使えるということです。例えば、<h1 id="heading">と書いた場合、同じページ内で"heading"というIDは再び使用できません。しかし、これがクラスなら何度でも使用できます。
IDはレイアウトの枠組みなどの変わることがない部分に、クラスは何度も使用する部分につけるなど使い分けましょう。また、IDとクラスを同時に指定した場合、IDが優先されるのも異なる点です。

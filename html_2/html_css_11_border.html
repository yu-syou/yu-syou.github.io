border-width

border-widthプロパティで線の太さを指定します。辺によって太さを変えたい場合には、上・右・下・左の順で変えてください。単位はremやpx、%です。
thin(細い線)、medium(普通の太さ)、thick(太い線)でも変えられます。

border-width: 1px;

border-width: thick;

border-width: 1px 4px 8px 12px;

border-style

border-styleプロパティで線の装飾方法を指定します。1つのスタイルを指定した場合、すべての辺にも適用されるので、辺によって変えたいときは 上・右・下・左の順で変えてください。
border-styleの初期値はnoneのため、border-widthとstyleは必ず一緒に記述する必要があります。以下にstyleの値を示します。

  • none　　線を非表示
  • solid　　 1本の実線
  • double　 2本の実線
  • dashed　破線
  • dotted　 点線
  • groove　立体的な谷型の線
  • ridge　　立体的な山型の線
  • inset　　囲まれた部分が凹んで見える立体的な線
  • outset　 囲まれた部分が浮き上がって見える立体的な線

border-style: solid;

border-style: double;

border-style: dashed;

border-style: dotted;

border-style: groove;

border-style: ridge;

border-style: inset;

border-style: outset;

border-style: dotted ridge inset outset;



border-color

border-colorプロパティを使って、線の色を指定できます。基本的な記述はstyleたちと一緒です。

border-color: #0bd;

border-color: tomato;

border-color: tan #0bd tomato #000



borderプロパティ

border-width,style,colorはまとめて書くことができます。border単体だと全ての辺に、top,left.right,bottomで辺を指定できます。

boder-bottom: 2px solid #0bd

border: 5px dotted tomato;

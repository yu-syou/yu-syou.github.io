Flexbox

レイアウトを組むとき、例えば画像やブロックを横配置したい時や、順序を変えて配置したい時があります。そんな時は、Flexboxという記述をとることでレイアウトの幅が広がります。

containerクラスにdisplay: flexが追加されたことで、横に並びました。プロパティはさまざまなものがあるので、以下に示します。



flex-direction

flex-directionプロパティ。並べる方向を縦、または横に指定できます。初期値はrow。(今はrow-severseとしています)

  • row　　　　　　　左から右に配置
  • row-severse　　 右から左に配置
  • column　　　　　上から下に配置
  • column-reverse　下から上に配置

flex-wrap

flex-wrapプロパティ。子要素を１行で並べるか、複数行で並べるかを指定できます。初期値はnowrapです。(今はwrap-reserveとしています)

  • nowrap　　　　　　子要素を折り返しせず、1行に並べる
  • wrap　　 　　　　 子要素を折り返し、複数行に上から下に並べる
  • wrap-reverse　　　子要素を折り返し、複数行に下から上に並べる

justify-content

justify-contentプロパティ。親要素(container)内に空きスペースがあった場合に、子要素を水平方向のどの位置に配置するかを指定します。(初期値はflex-start)今はspace-aroundとしています。
主な値は以下の通り。

  • flex-start　　　　　　　行の開始位置から配置。左揃え。
  • flex-end　　 　　　　　 行末から配置。右揃え。
  • center　　　　　　 　　中央揃え
  • space-between　　　　最初と最後の子要素を両端に配置し、残りの要素は均等に間隔を空けて配置
  • space-around　　 　　 両端の子要素を含め、均等に配置

align-items

align-itemsプロパティ。親要素に空きスペースがあった場合、子要素を垂直方向のどの位置に配置するかを指定します。(初期値はstretch)今はcenterとしています。

  • stretch　　　　　　　親要素の高さ、またはコンテンツの一番多い子要素の高さに合わせて広げて配置
  • flex-start　 　　　　親要素の開始位置から配置。上揃え
  • flex-end　　　　 　　　親要素の終点から配置。下揃え
  • center　　　　　　　　　中央揃え。
  • baseline　 　　 　　　ベースラインで揃える

align-content

align-contentプロパティ。子要素が複数行に渡った場合、垂直方向の揃えを指定します。しかし、flex-wrap: nowrap;が適用されていると無効になるので注意が必要です。初期値はstretchで、今はspace-aroundを指定しています。

  • stretch　　　　　　　親要素の高さに合わせて広げて配置
  • flex-start　 　　　　親要素の開始位置から配置。上揃え
  • flex-end　　　　 　　　親要素の終点から配置。下揃え
  • center　　　　　　　　　中央揃え。
  • space-between　 　　 　　最初と最後の子要素を上下の端に配置し、残りのようは均等に間隔を開けて配置
  • space-around　　　　上下端にある子要素も含めて、均等に間隔をあけて配置。
